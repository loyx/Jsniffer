# Jsniffer 是什么
![image](https://user-images.githubusercontent.com/30404367/170512376-079781b2-ed4f-4345-841d-03121b8caba4.png)

JSniffer 基于Jnetpcap的跨平台的抓包工具
# 项目结构
```text
Jsniffer:.
├─.idea
│  └─artifacts
└─src
    └─main
        ├─java
        │  └─cn/loyx/Jsniffer
        │      ├─capture
        │      │  └─Protocols
        │      │      ├─application
        │      │      ├─lan
        │      │      ├─network
        │      │      ├─tcpip
        │      │      └─wan
        │      ├─service
        │      └─ui
        └─resources
            ├─icons
            ├─lib
            └─META-INF
```
